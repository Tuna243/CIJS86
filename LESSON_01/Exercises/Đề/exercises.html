JavaScript Exercises Documentation

Exercise 1: Reverse a String

Write a function that takes a string as input and returns the reverse of that string.

Input:

A string

Expected Output:

The reverse of the input string.

Example:

reverseString('hello')

Output: 'olleh'

Exercise 2: Palindrome Checker

Write a function that checks whether a given string is a palindrome (reads the same forwards and backwards).

Input:

A string

Expected Output:

true if the input string is a palindrome, false otherwise.

Example:

isPalindrome('racecar')

Output: true

Write a program that prints the numbers from 1 to 100. For multiples of three, print "Fizz" instead of the number, and for multiples of five, print "Buzz". For numbers that are multiples of both three and five, print "FizzBuzz".

Exercise 4: Sum of Multiples

Write a function that takes an integer as input and returns the sum of all the multiples of 3 or 5 below that integer.

Input:

An integer

Expected Output:

The sum of all the multiples of 3 or 5 below the input integer.

Example:

sumMultiples(10)

Output: 23

Exercise 5: Prime Number Checker

Write a function that checks whether a given number is prime (divisible only by 1 and itself).

Input:

An integer

Expected Output:

true if the input number is prime, false otherwise.

Example:

isPrime(17)

Output: true

Exercise 6: Fibonacci Sequence

Write a function that generates the Fibonacci sequence up to a specified number of terms.

Input:

An integer representing the number of terms

Expected Output:

An array containing the Fibonacci sequence up to the specified number of terms.

Example:

fibonacci(8)

Output: [0, 1, 1, 2, 3, 5, 8, 13]

Exercise 7: Array Manipulation

Write a function that takes an array of numbers and returns a new array with all the numbers doubled.

Input:

An array of numbers

Expected Output:

A new array with all the numbers from the input array doubled.

Example:

doubleArray([1, 2, 3, 4])

Output: [2, 4, 6, 8]

Exercise 8: Object Manipulation

Write a function that takes an object as input and returns a new object with the keys and values swapped.

Input:

An object

Expected Output:

A new object with the keys and values swapped from the input object.

Example:

swapObject({ name: 'John', age: 30 })

Output: { 'John': 'name', '30': 'age' }

Exercise 9: Find the Longest Word

Write a function that takes a sentence as input and returns the longest word in the sentence.

Input:

A sentence

Expected Output:

The longest word in the input sentence.

Example:

findLongestWord('The quick brown fox jumps over the lazy dog')

Output: 'jumps'
